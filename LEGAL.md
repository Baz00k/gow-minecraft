# Legal & Usage Policy

## Summary

This project provides a **launcher container only**. No Minecraft game assets, binaries, or proprietary content are bundled with this image.

## Ownership Requirements

**Users are expected to own a legitimate Minecraft license for gameplay.**

This project does NOT provide:

- Minecraft game files
- Minecraft authentication credentials
- Methods to bypass Mojang/Microsoft authentication servers

## Offline/Local Profiles

Offline or local profiles in Prism Launcher are intended for:

- **Local/LAN play** — Single-player or local network multiplayer
- **Offline servers** — Servers configured with `online-mode=false`

Offline profiles do NOT require authentication but are limited to environments where online verification is not enforced by the server.

## What This Project Does NOT Do

- **No authentication bypass** — This image does not circumvent Mojang/Microsoft authentication systems
- **No piracy enablement** — This project does not endorse or facilitate unauthorized access to paid content
- **No bundled assets** — No Minecraft game files or proprietary content is included

## Prohibited Launchers

For security and legal reasons, the following launchers are **FORBIDDEN** and will NOT be supported:

| Launcher   | Reason                                       |
| ---------- | -------------------------------------------- |
| TLauncher  | Security concerns, unauthorized distribution |
| SKLauncher | Security concerns, unauthorized distribution |

Users attempting to integrate these launchers with this project will receive no support.

## Community Forks

Community forks of Prism Launcher that remove the MSA login gate (required before offline accounts become available in official Prism) are permitted. However:

- Users remain responsible for complying with Mojang/Microsoft terms of service
- This project takes no position on the legitimacy of specific forks
- No endorsement of any particular fork is implied
